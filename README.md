This is the start of something great.

Group members: Nathan Gabriel, Diego Prada, Adrian Rodarte, Kevin Gramajo

Alien W.O.L is an 3D FPS Shooter developed on Unity Editor. The primary objective of the game is to make your way towards the exit while shooting down enemies aboard the alien mothership. The story behind this game is that you are a rogue alien; gunning down other aliens after seeing the cruelty they treated humans with upon invading earth.
